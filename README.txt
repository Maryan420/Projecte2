###CONTROL INVENTARI###

Aquesta es una aplicació web per controlar els estocs de qualsevol empresa.

Seguint una sèrie de camps configurats en una base de dades, podreu inserir productes i consultar-los ja sigui de manera massiva o filtrant-los. 

Consta de 8 versions:

v1.0 - Versió inicial, base html no funcional
v1.2 - Estilització de l'aplicació
v1.3 - Primer contacte php
v1.4 - Enllaç amb base de dades i codi SQL
v1.5 - Eliminació html, nous estils
v1.6 - Funcionalitat registre i consulta productes finalitzada
v1.7 - Aplicat filtratge, per nom, ordre alfabètic, etc
v2.0 - Versió final, correcte funcionament, funcionalitat completa amb php
v2.1 - Versió final, correcte funcionament, funcionalitat completa amb php i css acabat
